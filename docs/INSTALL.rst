iii.theme Installation
----------------------

To install iii.theme using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``iii.theme`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        iii.theme

* Re-run buildout, e.g. with:

    $ ./bin/buildout
